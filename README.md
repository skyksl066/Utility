# Utility

## 簡介
Utility 是一套為 .NET 應用程式設計的實用工具與擴充功能套件。初期聚焦於 Dapper 擴充與屬性標註的資料表欄位對應，未來將持續擴充更多開發輔助功能，協助簡化資料存取、物件映射與日常開發工作。

## 特色
- 支援 Dapper 擴充功能
- 以屬性標註進行資料表欄位對應
- 提供常用的開發輔助工具
- 支援多版本 .NET（.NET Core 3.1, .NET 5, .NET 6, .NET 7, .NET 8, .NET 9）

## 快速開始

using Utility; // 依照文件使用 Dapper 擴充功能或其他工具

> 詳細用法請參考各功能的程式碼註解與未來的文件補充。

## 支援平台
- .NET Core 3.1
- .NET 5
- .NET 6
- .NET 7
- .NET 8
- .NET 9
